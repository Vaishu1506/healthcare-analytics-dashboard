# Healthcare Analytics Dashboard

## Overview

This project presents an interactive Healthcare Analytics Dashboard designed to analyze hospital and patient-related data through visual insights. The dashboard helps identify trends in patient admissions, billing amounts, bed occupancy, medication usage, and overall healthcare performance.

## Features

* Interactive filtering by medical condition
* Total patient and doctor statistics
* Bed occupancy analysis
* Billing amount overview
* Age group distribution
* Admission trends over time
* Medication score comparison

## Dashboard Insights

* Total Patients: 55K+
* Doctors: 40K+
* Beds Available: 11K+
* Billing Amount: 1.42B+
* Average Patient Age: 51 years

## Tools Used

* Microsoft Power BI
* Data Visualization Techniques
* Healthcare Dataset Analysis

## Key Visualizations

* Patient Admission Trends
* Billing Amount by Condition
* Age Group Analysis
* Medication Score by Medical Condition

## Project Objective

The objective of this project is to transform raw healthcare data into actionable insights for better decision-making in hospital management and healthcare operations.
